# Smart Home Guide 2025

Affiliate-Marketing Website für Smart Home Produkte mit Amazon Associates Monetarisierung.

## Features

- **SEO-optimiert**: Strukturierte Daten, Meta-Tags, Canonical URLs
- **Responsive Design**: Mobile-first Ansatz
- **Affiliatematic Integration**: AI-gesteuerte Produktempfehlungen
- **Moderne UI**: Gradienten, Animationen, Intersection Observer
- **Schnelle Ladezeiten**: Statisches HTML/CSS/JS

## Affiliate-Setup

1. **Amazon Associates Account**: Registriere dich bei [affiliate-program.amazon.com](https://affiliate-program.amazon.com)
2. **Affiliatematic**: Melde dich bei [affiliatematic.com](https://affiliatematic.com) an und füge deine Domain hinzu
3. **Tracking ID**: Ersetze `smarthomeguide0c-21` in `index.html` mit deiner eigenen Amazon Tracking ID

## Technik

- HTML5 semantisch
- CSS3 mit Custom Properties
- Vanilla JavaScript
- Keine externen Dependencies außer Affiliatematic
